# Java code repository maintained at VTI's Traffic Analysis and Logistics group

This repository contains open-source (GPL-licensed) code developed at or in collaboration with our group. Published packages are considered sufficiently stable for external use, but come with no warranty (see the GPL disclaimer for details). 

Currently available modules:





| Name | Description | Maintainer | Build/test |
|--------|---------------|----------------|--------------|
| ATAP | "Agent-based/atomic dynamic traffic assignment problem". A MATSim plugin for the approximation of Nash equilibria. | gunnar.flotterod@vti.se | [![ATAP](https://github.com/vtisweden/java-projects/actions/workflows/build-atap.yml/badge.svg)](https://github.com/vtisweden/java-projects/actions/workflows/build-atap.yml) | 
| CERTAIN | Simulates emergency service fleet performance, in particular with respect electrification parameters.  | gunnar.flotterod@vti.se | [![CERTAIN](https://github.com/vtisweden/java-projects/actions/workflows/build-certain.yml/badge.svg)](https://github.com/vtisweden/java-projects/actions/workflows/build-certain.yml)
| DYNAMEQ | Translate input files from Dynameq format to a format suitable for use in MATSim. | filip.kristofersson@vti.se | [![DYNAMEQ](https://github.com/vtisweden/java-projects/actions/workflows/build-dynameq.yml/badge.svg)](https://github.com/vtisweden/java-projects/actions/workflows/build-dynameq.yml) |
| EMULATION | A MATSim plugin that moves travelers and vehicles according to fixed travel times through the transport system. | gunnar.flotterod@vti.se | [![EMULATION](https://github.com/vtisweden/java-projects/actions/workflows/build-emulation.yml/badge.svg)](https://github.com/vtisweden/java-projects/actions/workflows/build-emulation.yml) |
| ROUNDTRIPS | Samples spatio-temporal population trajectories from a wide range of target distributions. | gunnar.flotterod@vti.se | [![ROUNDTRIPS](https://github.com/vtisweden/java-projects/actions/workflows/build-roundtrips.yml/badge.svg)](https://github.com/vtisweden/java-projects/actions/workflows/build-roundtrips.yml) |
| SAMGODS | Approximates and extends the Swedish national freight model system Samgods, in particular its logistics module. | gunnar.flotterod@vti.se | [![SAMGODS](https://github.com/vtisweden/java-projects/actions/workflows/build-samgods.yml/badge.svg)](https://github.com/vtisweden/java-projects/actions/workflows/build-samgods.yml) |
| TRAMODBY | Couples the Norwegian travel demand model TraMod_By to MATSim. | filip.kristofersson@vti.se | [![TRAMODBY](https://github.com/vtisweden/java-projects/actions/workflows/build-tramodby.yml/badge.svg)](https://github.com/vtisweden/java-projects/actions/workflows/build-tramodby.yml) |
| UTILS | General support functionality and tools | gunnar.flotterod@vti.se | [![UTILS](https://github.com/vtisweden/java-projects/actions/workflows/build-utils.yml/badge.svg)](https://github.com/vtisweden/java-projects/actions/workflows/build-utils.yml) |
