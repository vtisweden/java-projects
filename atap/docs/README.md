# ATAP - Agent-based Traffic Assignment Problem

The `se.vti.atap` package implements a solution heuristic for the agent-based traffic assignment problem. It contains the `se.vti.emulation` package that moves travelers and vehicles according to fixed travel times through the transport system. Both are MATSim plugins ("contribs"). For direct useability with MATSim (not relying on github packages), it also contains selected classes from the `se.vti.utils` package.

**To maintain consistency with ongoing MATSim developments, documentation and the most up-to-date version of this package are hosted on the MATSim github repository. [See here.](https://github.com/matsim-org/matsim-libs/tree/main/contribs/atap)**
